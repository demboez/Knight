 <h3>Installation</h3>
 
```bash
$ pkg install php
```

```bash
$ pkg install wget
```

```bash
$ pkg install openssh
```

```bash
$ git clone https://github.com/JamboHS/Knight.git
```

<h3>Usage</h3>

```bash
$ cd Knight
$ bash install.sh
$ bash knight.sh
 ```
 
 Captured images will be stored in the captured folder, if not type:

```bash
$ chmod +x copy.sh
$ ./copy.sh
```
